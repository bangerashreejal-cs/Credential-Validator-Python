# Credential Validator in Python

A simple Python application that validates user login credentials using custom logic. 

## Features
* **Phone Validation:** Ensures exactly 10 digits and no leading zeros.
* **Email Validation:** Checks for correct `@` format and domain length.
* **Password Security:** Enforces a mix of Uppercase, Lowercase, Digits, and Special Characters.
* **UX Animation:** Features a "Logging In..." terminal animation.

##  How to Use
1. Download the `main.py` file.
2. Run it in any Python environment:
   ```bash
   python main.py
